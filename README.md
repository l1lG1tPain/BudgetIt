# 🦈 BudgetIt 4.0.0 — Акулка на максималках

**Современное PWA-приложение** для учёта бюджета, расходов, вкладов и долгов.
Лёгкое. Красивое. Умное. Работает даже **оффлайн**.

Версия **4.0.0** — крупнейшее обновление за всю историю BudgetIt.
Новый навбар, планировщик, мощный поиск, импорт из банковских выписок и полная переработка аналитики.

---

# ✨ Что нового в версии 4.0.0

## ⚡ 1. Залётом — импорт из банковских выписок

Новая фича для тех, кто не хочет вводить транзакции вручную:

* Загрузи выписку в формате **Excel, CSV или PDF**
* Акулка сама определит тип операции, распарсит суммы и даты
* Умная авто-категоризация под узбекский рынок — сотни мерчантов уже в базе
* Поддерживаются: **Payme, Click, Humo, Uzcard, Uzum Bank, TBC, Kapitalbank, Ipak Yuli, Сбербанк, Т-Банк** и другие
* Превью транзакций перед импортом — видишь что получится до сохранения
* Возможность настроить маппинг колонок вручную

Просто закинь файл — Акулка сделает всё остальное 🦈

---

## 📆 2. Планировщик бюджета

Новая полноэкранная страница для планирования месяца:

* Задай ожидаемые доходы
* Добавь регулярные, разовые расходы и вклады
* Смотри на прогнозируемый остаток в реальном времени
* Планы сохраняются и доступны в любой момент
* Доступен из навбара и из настроек

---

## 🔍 3. Поиск по транзакциям

Мощный поиск по всей истории операций:

* Фильтрация по типу: доходы, расходы, вклады, долги
* Поиск по категории, сумме, названию товара
* Мгновенные результаты без перезагрузки

---

## 🧭 4. Новый навбар (Liquid Glass)

Полностью переработанная навигация:

* Четыре вкладки: **Бюджет, Аналитика, Планировщик, Профиль Акулки**
* Стиль Liquid Glass с анимацией
* Стеклянные SVG-иконки в едином стиле
* **Зажми «Бюджет»** — появится быстрое переключение между бюджетами

---

## 📊 5. Переработанная аналитика

Графики и инсайты получили полный редизайн:

* Градиентные линейные чарты с заливкой
* Единая цветовая палитра под тему оформления — цвета берутся из CSS-переменных
* Скруглённые бары, тепловая карта по дням недели
* Инсайты: иконки в карточках, анимация health-bar при каждом открытии
* Кнопка закрытия для информационных уведомлений
* Подзаголовки к каждому графику

## 👤 6. Профиль Акулки

Переработаны страницы профиля, достижений и темы оформления:

* Новая навигация между подстраницами
* Обновлены достижения и условия их открытия

# 🛠 Под капотом

* Исправлена **XSS-уязвимость**: пользовательские данные в `innerHTML` теперь проходят через `escapeHtml`
* **Umami аналитика** полностью переписана: очередь событий до загрузки трекера, трекинг SPA-навигации, все ключевые действия
* **Service Worker** обновлён до `v4.0.0`
* Исправлены коллизии ID при массовом импорте транзакций
* Улучшено автоопределение колонок при импорте (узбекские термины `sana`, `miqdor`, `kirim`, `chiqim`)
* Исправлены ложные срабатывания regex в категоризаторе (word boundaries)

# 🛠 Технологический стек

* PWA + офлайн-режим
* IndexedDB (fallback: localStorage)
* ES-модули (Vanilla JS)
* Chart.js

* Service Worker
* Темы на CSS Variables
* Custom UI-компоненты
